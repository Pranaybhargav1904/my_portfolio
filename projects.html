AI-Powered Expense Tracker

A full-stack Next.js application that helps users track and analyze their expenses. Users can create accounts, categorize expenses, and interact with an intelligent AI assistant powered by Groq's Llama 3.3 model to ask natural language questions about their spending patterns. The backend uses Supabase for PostgreSQL database and authentication, while the front-end features a modern dashboard built with React 19 and Tailwind CSS. The AI assistant can execute various tools to retrieve expense data, generate summaries, and provide personalized financial insights through conversational chat.

Smart Resume Analyzer

A web application that allows users to upload their resumes and job descriptions to receive AI-generated insights. The system analyzes the resume, identifies skill gaps, and provides personalized improvement suggestions. It also calculates a match score showing how well the resume aligns with the job description provided. The goal is to help users optimize their resumes for better job application results.
